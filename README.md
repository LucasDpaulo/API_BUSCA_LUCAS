# API_BUSCA_LUCAS

API de Automação para Extração de Dados Hinova e Disparo via Quepasa.

Sistema SaaS simplificado para monitoramento diário de ativos, vendas e saúde financeira, com entrega automatizada de relatórios via WhatsApp.

## Visão Geral

Este projeto automatiza a coleta de dados financeiros da API Hinova, processa cálculos do mês corrente e envia relatórios formatados via WhatsApp através da API Quepasa.

## Tecnologias

- **Linguagem:** Python 3.11+
- **APIs:** Hinova (coleta de dados) | Quepasa (envio WhatsApp)
- **Agendamento:** Execução automática diária às 19:00
